# ITI Docker Labs 

**System Administration Track - Intake 46**

Welcome to my Docker repository! This workspace contains my practical lab submissions for the Docker and Containerization module.

## Repository Structure

The repository is organized into separate directories for each lab assignment. Each folder contains its own detailed `README.md` with the specific commands used, configuration files (like `Dockerfile` and `docker-compose.yml`), and verification screenshots.

* 📁 **[Lab 2](./lab-2/)**: 
  * Containerizing a Python Flask application.
  * Pushing and pulling images from Docker Hub.
  * Configuring custom bridge networks and volumes using Nginx.

* 📁 **[Lab 3](./lab-3/)**: 
  * Setting up and configuring a local insecure Docker registry (CNCF Distribution).
  * Building custom Alpine-based images and pushing them to the local registry.
  * Orchestrating multi-container applications (WordPress & MySQL, Flask) using Docker Compose.
